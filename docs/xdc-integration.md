# 🔄 Blockticity x XDC Network

Blockticity integrates with the XDC Network to enable gasless NFT redemptions and walletless claims for verified real-world assets (RWAs).

---

## ✅ Why XDC?

- 🌐 XDC is an EVM-compatible L1 tailored for trade finance and enterprise-grade tokenization.
- ⚙️ Blockticity uses XDC’s Apothem testnet to pilot gasless COA redemption via relayer.
- 🔒 Wallets are abstracted using Privy for smooth onboarding — no browser extension needed.

---

## 🧪 Testnet Pilot

We successfully completed:

- ✅ Gasless claim flow using Apothem testnet  
- ✅ Privy-authenticated redemption of pre-minted NFTs  
- ✅ SKU-mapped metadata resolution + COA verification
